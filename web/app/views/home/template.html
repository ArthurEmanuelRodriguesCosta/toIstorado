Sobre o projeto

O Tô Estourado quer, através de uma temática lúdica, despertar no cidadão o sentimento
de vigilância para com o bem público, no sentido de facilitar a identificação de possíveis
irregularidades - neste caso, o enriquecimento ilícito dos candidatos.

Este projeto surgiu no II Hackfest contra a corrupção promovido pelo Ministério
Público do estado da Paraíba e Laboratório Analytics.

Os créditos deste projeto são de Arthur Costa, Glauco PorDeus, Marianne Linhares,
Talita Lôbo e Victor Almeida.
Contribuições e sugestões são muito bem-vindas e podem ser feitas a partir do GitHub
do projeto, onde também estão disponíveis o código e dados utilizados.
